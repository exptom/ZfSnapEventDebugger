ZfSnapEventDebugger
===================

Event Debugger for ZendDeveloperTools in Zend Framework 2

Instalation
-----------

Add dependencies to composer.json:

```json
{
    "require-dev": {
        "snapshotpl/zf-snap-event-debugger": "dev-master"
    }
}
```

And add module `ZfSnapEventDebugger` in `appilcation.config.php`
